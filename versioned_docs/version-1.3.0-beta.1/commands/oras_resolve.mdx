# oras resolve

[Experimental] Resolves digest of the target artifact

```bash
oras resolve [flags] <name>{:<tag>|@<digest>}
```

## Examples

Resolve digest of the target artifact:

```bash
oras resolve localhost:5000/hello-world:v1
```
