# oras discover

[Preview] Discover referrers of a manifest in a registry or an OCI image layout
** This command is in preview and under development. **

```bash
oras discover [flags] <name>{:<tag>|@<digest>}
```

## Examples

Discover referrers of manifest 'hello:v1' in registry 'localhost:5000', displayed in a tree view:

```bash
oras discover localhost:5000/hello:v1
```

Discover referrers via referrers API:

```bash
oras discover --distribution-spec v1.1-referrers-api localhost:5000/hello:v1
```

Discover referrers via tag scheme:

```bash
oras discover --distribution-spec v1.1-referrers-tag localhost:5000/hello:v1
```

[Experimental] Discover referrers and display in a table view:

```bash
oras discover localhost:5000/hello:v1 --format table
```

[Experimental] Discover referrers and format output with Go template:

```bash
oras discover localhost:5000/hello:v1 --format go-template --template "{{.manifests}}"
```

Discover all the referrers of manifest with annotations, displayed in a tree view:

```bash
oras discover -v localhost:5000/hello:v1
```

Discover referrers with type 'test-artifact' of manifest 'hello:v1' in registry 'localhost:5000':

```bash
oras discover --artifact-type test-artifact localhost:5000/hello:v1
```

Discover referrers of the manifest tagged 'v1' in an OCI image layout folder 'layout-dir':

```bash
oras discover --oci-layout layout-dir:v1
```
