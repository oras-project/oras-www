# oras logout

Log out from a remote registry.

```bash
oras logout [flags] <registry>
```

## Example

Log out from a remote registry.

```bash
oras logout localhost:5000
```

## Options

```
  -d, --debug                         debug mode
  -h, --help                          help for logout
      --registry-config stringArray   auth config path
```
